<h2>Como utilizar</h2>
<p>Para clonar e executar este aplicativo em seu computador, você precisará seguir as etapas abaixo</p>

```
# Clonar repositório
  $ git clone https://github.com/leovenom/happy-app.git

# Acessar o diretório
  $ cd happy/web
  $ cd happy/backend

# Instalar as depedências
  $ yarn install
  
  Aproveite
```
